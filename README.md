# icecream 2.0
**아이스크림 주문 포스기 2.0** 입니다.<br><br><br>

### 개발 기간
* 2024.01.22 ~ 2024.01.30 (8일)<br><br>

### 개발 환경
* Java 11
* **프레임워크** : Spring Boot(2.7.17)
* **데이터베이스** : Mysql - MyBatis
* **프론트** : thymeleaf, javascript, jQuery<br><br>
